== Description ==
jMarquee is an easy Pre-packed jQuery News Ticker plugin with ready to use styles.

<h4>Features</h4>
* jMarquee - jQuery News Ticker:
	jMarquee is an easy Pre-packed jQuery News Ticker plugin with ready to use styles.
	
== Installation == 
* In your Admin, go to menu Plugins > Add
* Search for jMarquee - jQuery News Ticker
* Click to install
* Activate the plugin
